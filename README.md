## Getting Started

You need php 8.0 

### Installation

1. Clone the repository `git clone`
2. Create and fill `.env` file from `.env.example`
3. Run `composer install`
4. Run `php artisan key:generate`
5. Run `php artisan migrate --seed`

## Testing

`php artisan test`
